BMI Calculator API Documentation

Created By Subham Kumar Sinha

API Endpoint

Method: GET

Endpoint: /api/bmi/{weight}/{height}

Framework Used: Flask (Python)

Example Request

Request URL: http://localhost:port/api/bmi/87.9/1.75

Path Parameters

Parameter Type Required Description
weight float Yes Body weight in kilograms
height float Yes Height in meters

Response

Success (200 OK)

{
  "Category": "Overweight",
  "bmi": 28.702,
  "height": 1.75,
  "weight": 87.9
}

Fields:

  • Category: Health category based on BMI
  • bmi: Calculated BMI (trimmed to 3 decimal points)
  • height: Provided height in meters
  • weight: Provided weight in kilograms

Error Responses

1. Height is Zero (400 Bad Request)

{
  "Error": "Height parameter cannot be Zero"
}

2. Invalid Parameters (400 Bad Request)

{
  "error": "Invalid parameters. Please provide numeric values for weight and height."
}

BMI Categories Reference

BMI Range Category
< 18.5 Underweight
18.5 – 24.9 Normal weight
25 – 29.9 Overweight
30 and above Obese
